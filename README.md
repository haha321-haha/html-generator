HTML转图片工具

一个简单易用的HTML转图片工具，基于html2canvas实现，能够将HTML内容转换为高质量的PNG图片，完美去除多余白边，特别适合内容创作者和开发者使用。

✨ 功能特点

• 智能去白边 - 自动检测内容边界，去除多余空白

• 高质量输出 - 支持高清渲染（2倍缩放）

• 实时预览 - 编辑HTML代码时实时查看效果

• 一键下载 - 生成后可直接下载PNG图片

• 响应式设计 - 适配各种设备屏幕尺寸

• 完全免费 - 基于MIT开源协议，无任何限制

🚀 快速开始

在线使用

直接访问 https://haha321-haha.github.io/html-generator/html2image_tool.html 即可使用。

本地使用

1. 克隆仓库：
git clone https://github.com/haha321-haha/html-generator.git


2. 打开 html2image_tool.html 文件：
cd html-generator
# 直接在浏览器中打开html2image_tool.html文件


📖 使用方法

1. 输入HTML代码 - 在左侧编辑框中编写或粘贴HTML代码
2. 实时预览 - 点击"预览HTML"查看渲染效果
3. 生成图片 - 点击"生成图片"转换为PNG格式
4. 下载保存 - 使用"下载PNG图片"保存到本地
